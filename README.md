# TypeError

Open source cybersecurity engineering focused on practical tools, useful metrics, and data driven security work.

TypeError spans application security, vulnerability management, security metrics, and security data science. The focus is operational clarity: building software and supporting material that help teams measure risk, explain tradeoffs, and make better decisions.

This repository contains the static homepage for [typeerror.com](https://typeerror.com/).

## What TypeError Is

An open source security engineering effort for people who need security work to be understandable and usable in practice.

The goal is not to publish noise. It is to build and share tools, experiments, and reference material that turn messy security inputs into clearer signal.
